# Apan-Intern-Job-Search-Website

A modern, high-end job search platform designed for students and companies. Features a completely redesigned user interface with premium aesthetics, a clean layout, and a smooth user experience.

## Features
- **Redesigned UI**: Modern, high-end theme with premium typography (Clash Display, Satoshi).
- **Student Dashboard**: Browse internships and jobs with ease.
- **Company Tools**: Manage listings and applications efficiently.
- **Responsive Design**: Optimized for all devices.

## Tech Stack
- **Backend**: Node.js, Express.js
- **Frontend**: EJS, JavaScript, Vanilla CSS
- **Database**: MongoDB (Mongoose)

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the server: `npm start`.
4. Open `http://localhost:8080`.
